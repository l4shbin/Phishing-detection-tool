# Phishing-detection-tool 🛡️

A professional heuristic-based URL analyzer designed for the Termux environment. Built with a focus on speed, minimalism, and terminal efficiency.

## 📁 Repository Structure
* `src/main.py` - Core analysis engine.
* `setup.sh` - Automated environment setup.

## 🚀 Installation
Run the following commands in Termux:

```bash
git clone [https://github.com/l4shbin/Phishing-detection-tool](https://github.com/l4shbin/Phishing-detection-tool)
cd Phishing-detection-tool
bash setup.sh
python src/main.py
